# Vibe-Caster - Musik efter väder

**Vibe-Caster** är en interaktiv webbapplikation (mashup) som automatiskt genererar en spellista efter aktuellt väder på vald plats.

---

## ✨ Funktioner
**Global vädersökning:** Hämtar realtidsdata om temperatur och väderförhållande
**Interaktiv karta:** Visar den sökta platsen med Leaflet och OpenStreetMap.
**Väderstyrd musik:** En specialbyggd algoritm matchar vädret mot specifika spellistor på Deezer.
**Dynamiskt tema** Bakgrundsfärger och animationer ändras baserat på väder.

## 🛠️ API:er
* [OpenWeatherMap](https://openweathermap.org/) (Väderdata)
* [Leaflet](https://leafletjs.com/) (Kartor)
* [Deezer API](https://developers.deezer.com/) (Musik)
